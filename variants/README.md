# Variants

Variants are optional stack add-ons you can apply when creating a new project.

Available:
- `python-fastapi` (working API starter)
- `node-ts` (working TS starter)
- `nextjs` (default web stack guidance + Makefile + CI scaffold)

Usage:
```bash
./scripts/new-project.sh ~/code/my-project "My Project" python-fastapi
```
